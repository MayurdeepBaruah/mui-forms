# Dependencies

Dependencies use to define relationships and dependencies between different fields

## Properties

| Name         | Description                                        |
| ------------ | -------------------------------------------------- |
| exists       | Checks criteria whether to display field in the UI |

### Properties for exists dependency

| Name         | Description                                        |
| ------------ | -------------------------------------------------- |
| ref          | It is the reference name of the  dependent field |
| value        | It is the value of the reference field. <br/>The current field will be displayed only if the reference field has the value specified here |
