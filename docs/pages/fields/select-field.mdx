## Field Properties

| Name        | Description                                |
| ----------- | ------------------------------------------ |
| displayType | The displayType for this field (`select`)  |
| displayName | The display name used for this field       |
| options     | An list of available options for the field |
| value       | The value used by this field               |
| placeholder | The hint value used for this field         |
| isDisabled  | Marks the field as disabled                |
| isReadonly  | Marks the field as readonly                |

## Config 
| Name        | Description                                |
| ----------- | ------------------------------------------ |
| type | The displayType for this field (`select`)  |
| url | The base api endpoint url |
| pathParams | The path parameters |
| queryParams | List of query parameters|
| valueKey | The object key of the response from which field value need to be mapped |
| labelKey | The object key of the response from which field display name needs to be mapped|

### An example config that fetches *state options* from a remote api depending on the country field value:

```json
"config": {
    "type": "url_loader",
    "url": "/country/",
    "pathParams": [["state"]],
    "queryParams": [["country", {"type": "fieldValue", "ref": "country"}]],
    "valueKey": "state#id",
    "labelKey": "state#name"
}
```
